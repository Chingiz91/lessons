cd name - перемещает наc в какую-либо папку (директорию)
ls - просматривает содержимое папки (деректории)
touch name -  создает файл
mkdir name - создает папку (директорию)


ls -a - показывает файлы и папки вместе со скрытыми файлами и папками
ls -l - показывает файлы и папки с доп информацией
ls -la - показывает файлы и папки со скрытыми файйлами и папками + доп информацию


cd - перемещает в домашнюю папку
cd ~ - переходит в домашнюю папку
cd . - переходит в ту папку в которой вы находитесь
cd .. - переходит в предыдущую папку (родительскую)


rm name - удаление файла name
rmdir - удаление ПУСТОЙ папки (дериктории)
rm -rf - удаляет все: папки с содержимым, файлы, пустые папки

-
history - выводит историю команд
history (число) - выводит историю в набранном количестве
history | grep (слово) - выводит команды, в которых есть указанное слово


pwd - показывает абсолютный (начиная с корневой папки) путь до текущей территории в которой вы находитесь
cat (название файла)  - выводит содержимое файла
nano (название файла)  - текстовый редактор, при помощи которого мы можем записать данные в файл


mv test1.txt test3.txt - переименрвывает файл test1.txt на test3.txt
mv test1 test3 - переименовывает папку test1 в test3
mv test1.txt test - перемещает файл test1.txt в папку  test


apt - системный установщик пакетов
snap -  установщик пакетов (чаще всего используют  для уствновки программ)
sudo (superuser do) - выполнение какой либо команды от имени администратора
 

 echo - записывает строку в какой либо файл
